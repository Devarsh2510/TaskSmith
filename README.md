# 🚀 TaskSmith

## 📌 Overview
TaskSmith is a full-stack task management application with a frontend and backend system.

## 🏗 Project Structure

tasksmith/
├── frontend/  (Expo / React Native)
├── backend/   (Your backend code)

## 🛠 Tech Stack
- Frontend: React Native (Expo)
- Backend: (Node.js / Flask / etc)
- Database: (MongoDB / MySQL)

## ⚙️ Setup Instructions

### 🔹 Frontend
cd frontend
npm install
npx expo start

### 🔹 Backend
cd backend
npm install
npm run dev

## ✨ Features
- Task creation & management
- User authentication
- API integration
